Permis de feu — Travaux par points chauds
Ce permis de feu doit être établi avant le démarrage de tout travail utilisant un outil susceptible de générer un point chaud (flamme nue, arcs électriques, étincelles).
INTERDICTION : Les travaux par points chauds sont strictement interdits après 12h00 (midi). Tout travail par point chaud doit être terminé avant midi afin de permettre les rondes de surveillance post-intervention.
OBLIGATION : Ce permis de feu doit être obligatoirement signé et validé par le client (Entreprise Utilisatrice) avant le début de toute réalisation de travaux par points chauds. Aucun travail ne peut débuter sans cette validation.
DURÉE MAXIMALE : Le permis de feu ne peut pas excéder 24 heures. Au-delà de cette durée, un renouvellement avec nouvelle validation de l'EU est obligatoire pour chaque période de 24h.
Le permis de feu est obligatoire pour tout travail par point chaud. Il doit être renouvelé quotidiennement.

Aide — Guide d'utilisation

Suivi PDP est une application de gestion des Plans de Prévention (PDP), des Inspections Communes Préalables (ICP) et des Permis de Feu pour Agriwatt et Enersolys.

L'application fonctionne hors-ligne : toutes les données sont stockées localement dans votre navigateur (IndexedDB). Elle est accessible depuis un ordinateur ou un smartphone.

  • Tableau de bord : vue d'ensemble des PDP actifs, expirés, ICP récentes
  • PDP : création, suivi, export (Word, PDF, ZIP)
  • ICP : visite préalable avec risques, photos, signatures
  • Entreprises : répertoire partagé d'entreprises extérieures
  • Paramètres : configuration EU par défaut, sites, import/export

  1. Cliquez sur Nouveau PDP depuis le tableau de bord ou le menu
  2. Remplissez les onglets dans l'ordre suggéré par la barre de progression en haut
  3. Général : titre, numéro, dates, site, description
  4. Type de chantier : choisissez "Chantier INTERNE" (dans nos locaux, EU = Agriwatt/Enersolys) ou "Chantier EXTERNE" (chez un client, le client devient l'EU)
  5. Entreprises : renseignez l'EU et ajoutez les EE intervenantes
  6. Visite : liez ou créez une ICP
  7. Risques : utilisez le sélecteur de risques par catégorie
  8. Mesures : ajoutez les mesures de prévention (boutons rapides ou manuellement)
  9. Urgence : contacts et numéros d'urgence du site
  10. Signatures : signature de l'EU et de chaque EE
  11. Cliquez sur Enregistrer à tout moment

Astuce : utilisez les boutons Précédent / Suivant en bas de chaque section pour naviguer.

L'application intègre une base de données de sites prédéfinis (Agriwatt Bouxières, Seclin, Vervins, Honfleur, Enersolys Bailly-Romainvilliers).

  • Dans un PDP, section Entreprises : le sélecteur "Sélection rapide du site" remplit automatiquement les infos EU (raison sociale, adresse, responsable, téléphone...)
  • Dans une ICP, section Participants : même sélecteur pour pré-remplir le représentant EU
  • Chaque site peut avoir un responsable / conducteur de travaux différent
  • Gérez vos sites dans Paramètres > Sites Agriwatt / Enersolys (ajouter, modifier, supprimer)

Les entreprises extérieures sont gérées via un répertoire partagé.

  • Quand vous ajoutez une EE manuellement dans un PDP, elle est automatiquement enregistrée dans le répertoire pour réutilisation future
  • Lors de l'ajout suivant, le répertoire vous propose les EE connues
  • Vous pouvez aussi gérer les EE depuis l'onglet Entreprises du menu principal
  • Import CSV Sellsy possible depuis l'onglet Entreprises
  • Les sous-traitants sont rattachés à leur EE principale
  • Modifier une EE dans un PDP met à jour le répertoire partagé

L'ICP (ex-VIC) est la visite préalable obligatoire avant tout travail sur site (art. R4512-2 du Code du travail).

  • Créez une ICP depuis le menu ou directement depuis un PDP (onglet Visite)
  • Renseignez : date, lieu, participants EU/EE, risques identifiés avec localisation sur site, observations
  • Ajoutez des photos prises sur le terrain (preuves visuelles)
  • Faites signer les participants (signature tactile) pour preuve formelle de réalisation
  • Liez l'ICP à un PDP existant ou créez un PDP depuis l'ICP
  • Les risques identifiés en ICP peuvent être envoyés directement vers le PDP lié
  • Export ICP autonome : PDF et Word avec toutes les données, photos et signatures
  • Intégration dans le PDP : quand une ICP est liée, elle apparaît automatiquement en chapitre 3 de l'export PDP (Word/PDF) avec participants, risques terrain, photos et signatures

Le permis de feu est intégré directement dans le PDP (onglet Permis de Feu).

  • Activez-le avec le bouton "Activer le Permis de Feu"
  • Sélectionnez les types de travaux (soudage, meulage, etc.)
  • Renseignez les risques, dates/horaires de validité, opérateurs
  • Cochez les consignes avant / pendant / après les travaux
  • 3 signatures : EU, EE intervenante, Exécutant
  • Le permis de feu est inclus dans l'export complet du PDP
  • Important : le permis doit être signé par l'EU (notre représentant en interne, ou le client en externe) avant tout travail

Chaque PDP et ICP peut être exporté en plusieurs formats :

  • Word (.docx) : document éditable avec mise en forme professionnelle, logo, tableaux
  • PDF : document figé pour impression ou envoi
  • ZIP complet (PDP uniquement) : le document principal + tous les fichiers joints en annexe
  • Impression : bouton imprimer disponible pour impression directe

Structure de l'export PDP (Word/PDF) :

  1. Identification des parties (EU / EE)
  2. Objet de l'intervention
  3. Inspection Commune Préalable (ICP) : si une ICP est liée, ses données complètes sont intégrées (participants, risques identifiés sur site avec localisation, observations, photos, signatures des participants)
  4. Risques et mesures de prévention (seules les mesures cochées sont exportées)
  5. Exigences EU
  6. Consignes d'urgence
  7. Validation et signatures
  8. Émargement
  9. Pictogrammes et règles de sécurité (pictogrammes visuels normés INRS : interdictions en rouge, EPI obligatoires en bleu)
  10. Permis de feu (si activé)

  • Le numéro de version du PDP s'incrémente automatiquement à chaque ajout de modification/avenant (pas à chaque enregistrement)
  • Ajoutez des modifications via l'onglet Modifications (type, description, auteur)
  • Chaque modification est tracée dans l'historique
  • Supprimer une modification fait revenir le numéro de version en arrière
  • L'historique peut être réinitialisé si nécessaire

  • Passez le statut d'un PDP à Clôturé quand les travaux sont terminés
  • Un PDP clôturé passe en lecture seule : tous les champs sont verrouillés
  • Une bannière rouge indique le verrouillage
  • Le bouton Réouvrir permet de repasser le PDP en statut Actif pour le modifier à nouveau
  • Les exports restent disponibles même sur un PDP clôturé
  • Clôture et réouverture sont tracées dans l'historique

Les données sont stockées localement dans votre navigateur. Elles ne sont pas synchronisées automatiquement entre appareils.

  • Exporter les données (JSON) : télécharge un fichier JSON contenant tous vos PDP, ICP, entreprises, sites et paramètres
  • Exporter l'app avec données : génère un fichier HTML autonome contenant l'application complète + toutes vos données. Idéal pour partager avec un chef de chantier ou un sous-traitant sans connexion. Le destinataire ouvre le fichier dans son navigateur et retrouve toutes les données (PDP, ICP liées, risques, photos, signatures)
  • Importer (ajouter) : fusionne les données importées avec les données existantes
  • Importer (remplacer tout) : efface les données existantes puis importe — idéal pour transférer d'un appareil à un autre
  • Réinitialiser : supprime toutes les données (les sites par défaut sont recréés)

Pour partager avec un collaborateur : utilisez "Exporter l'app avec données" — un seul fichier HTML suffit, pas besoin de connexion internet ni de compte.

Pour transférer entre appareils : Exportez les données (JSON) sur l'ancien, puis Importez (remplacer tout) sur le nouveau.

Risques :

  • Le sélecteur de risques propose un catalogue classé par catégorie (Chutes, Électrique, Incendie, Chimique, Mécanique, Circulation, Environnement, Travaux spécifiques)
  • Chaque risque a un niveau de gravité : Faible, Moyen, Élevé, Critique
  • Les mesures de prévention sont présentées sous forme de checkboxes individuelles : cochez uniquement celles applicables à votre situation. Les mesures non cochées n'apparaissent pas dans les exports
  • Vous pouvez ajouter des mesures personnalisées via le bouton "+" sous chaque risque
  • Vous pouvez ajouter des risques personnalisés (ex: "Accès toiture par crinoline")

Pictogrammes (normés INRS / ISO 7010) :

  • L'onglet Pictogrammes affiche les panneaux de sécurité visuels normés
  • Interdictions (cercle rouge, barre diagonale) : vapotage, vitesse, accès, flamme, feu, entreposage, stimulateur cardiaque
  • EPI obligatoires (cercle bleu, symbole blanc) : casque, chaussures, téléphone, vêtement, gants, protection antichute
  • Les pictogrammes cochés apparaissent en grille visuelle avec images dans les exports Word et PDF

  • La barre de progression en haut du PDP indique le taux de complétion
  • Les coches vertes sur les onglets indiquent les sections complétées
  • Le point vert/orange à côté de "Enregistré" indique si des modifications non sauvegardées existent
  • Sur mobile, utilisez le bouton + en bas pour créer rapidement un PDP ou une ICP
  • L'application fonctionne hors-ligne grâce au service worker
  • L'URL : https://cocothib.github.io/suivipdp/
